# Pet Vaccination System

The system is designed to manage and track the vaccination records of pets.

## Features

- Add new pets to the system
- Record vaccination details for each pet
- View and update vaccination records
- Generate vaccination reports

## Environment Variables

- `NEXT_PUBLIC_SUPABASE_SERVICE_ROLE_KEY`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `NEXT_PUBLIC_SUPABASE_URL`
